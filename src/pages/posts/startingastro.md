# Fuwari Blog

Fuwari is a minimal, fast, and beautiful blog template built with [Astro](https://astro.build/) and [Pagefind](https://pagefind.app/). This repository is a fork that you can customize and deploy easily.

---

## Features

- Static site generated with Astro
- Search powered by Pagefind
- Supports social icons via Iconify
- Deployable on Vercel
- Optimized for speed and SEO

---
